Начало работы

Быстрый старт

1. Скачать и установить

Скачайте оба файла с страницы загрузок:

  • DeepAtom.exe (135 МБ)
  • yara_rules_combined.yar (5.83 МБ)

Поместите их в одну папку и запустите DeepAtom.exe.

2. Первое сканирование

Откроется GUI с тремя вкладками: Сканер, Карантин и Защита.

1. Выберите папку или файл для сканирования
2. Выберите режим сканирования (Быстрый/Обычный/Глубокий)
3. Нажмите "Начать сканирование"

3. Включить защиту в реальном времени

Перейдите на вкладку "Защита" и включите Real-Time Guard для постоянной защиты.
