  • 500 MB free disk space

Recommended

Setup

Installation in 4 steps

1

Install Python 3.8+

Download and install Python from the official source, and make sure it's added to your PATH.

https://www.python.org/downloads/
2

Install dependencies

From the DeepAtom folder, install the required packages.

pip install -r requirements.txt
3

Enable Proactive Defense (optional)

Adds behavioral monitoring on top of the core scanner.

pip install wmi watchdog psutil pywin32
4

Launch DeepAtom

Start the application and run your first scan.

python DeepAtom.py
