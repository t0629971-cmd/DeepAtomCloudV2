DeepAtom is an advanced machine learning-powered antivirus for Windows. It combines static analysis, behavioral monitoring, and proactive defense to detect and block malware — with a particular focus on Windows PE (executable) threats.
Install Python 3.8+, run pip install -r requirements.txt, then launch python DeepAtom.py. See the Download page for the full step-by-step guide, including the optional Proactive Defense dependencies.
Minimum: Windows 7 (x64), dual-core 2.0 GHz CPU, 2 GB RAM, 500 MB free disk space. Recommended: Windows 10/11 (x64), quad-core 2.5+ GHz CPU, 4 GB RAM, and an SSD.
Yes. DeepAtom is free for personal and commercial use.
DeepAtom extracts 15 static features from each PE file and feeds them into a LightGBM RandomForest classifier trained on thousands of labeled samples, reaching 85–90% accuracy on known malware with a prediction time of roughly 50ms per file.
Proactive Defense is a Ring-3 behavioral analysis system that monitors processes, files, and the registry in real time to detect ransomware, code injection, antivirus tampering, and persistence mechanisms — even when no known signature exists.
85–90% for known PE malware, 80–85% for packed files, and 20–50% for zero-day threats. DeepAtom specializes in PE file analysis; see the full detection breakdown on the Features page.
Yes — DeepAtom has an open architecture. You can retrain the ML model on your own dataset, add or edit YARA rules, adjust the behavioral rules used by Proactive Defense, and modify the codebase directly. See the Training the ML Model guide.
Yes. All scanning and analysis happens locally on your machine. The optional cloud feature only ever transmits file hashes, never files themselves, and can be disabled entirely in Settings.
YARA AutoGen automatically creates new rules from confirmed detections and reloads the scanner without a restart. You can also manually update the SHA256 blacklist and YARA rule files at any time.
